Teamwork App
============

The Teamwork App is a project management tool for you and your team.
it allows you to create projects and assign specific tasks to your team members.
It pulls data from mongo db and shows all the team members what stage each part of 
the project is on and has a *simple*n and *intuitive* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.
